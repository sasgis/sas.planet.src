Немного истории версий кэша МЯК:

v1 - файлы .ogf2 - использовался в SmartComGPS и МЯК v1.xx для Symbian,
     java, WM. Самая первая версия Мобильных Яндекс.Карт.

v2 - файлы .ogf3 - использовался в МЯК v2.xx для Symbian, java, WM
     легендарная линейка версий программы, когда программа набрала
     нынешнюю популярность и множество функций. Но было ограничение -
     1 файл - 1 город. В программе нужно было в меню выбирать город из
     очень короткого списка. Трасс между городами также не было.

v3 - используется в линейке версий МЯК v3.xx для Symbian, java, WM, Bada,
     Maemo, Android (до 1.61). Формат хранения карт совершивший революцию
     в развитиии программы - в структуре папок файлы со специальными именами
     покрывают квадратами всю землю, независимо от населённых пунктов -
     покрытие идёт по координатам GPS.

V4 - Symbian v3.91..v3.96, Android v1.72..v3.99 - карты хранят дополнительную
     информацию о версии каждого тайла, таким образом при обновлении карт,
     даже мелком, как например в случае с Народной картой, редактируемой
     пользователями - из сети загружаются и обновляются только действительно
     изменившиеся тайлы карт.
